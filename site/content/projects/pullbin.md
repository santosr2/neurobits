Pullbin is a web scraping tool built with Python and BeautifulSoup for extracting content from paste sites like Pastebin and Ghostbin.

## What it does

Given a paste key, Pullbin fetches and extracts the content from supported paste sites. Useful for security research, CTF challenges, and automated content collection.

## Technologies

- **Python** for the core implementation
- **BeautifulSoup** for HTML parsing
- **Requests** for HTTP operations

## Status

This was an early project built for fun and learning. It's archived but the code remains available for reference.

See the [GitHub repository](https://github.com/santosr2/pullbin) for the source code.
